Smart Placement Tracker
A fullstack web application built with Spring Boot (Java) and React (JavaScript) to track student placements in college.

Technologies Used
Java Spring Boot (Backend REST APIs)

React (Frontend UI)

Spring Data JPA (Database access)

H2 Database (Testing) / MySQL/Postgres (Production)

Axios (API calls)

Chart.js / Recharts (Analytics & Charts)

Features
Student Management (CRUD operations)

Company Management (CRUD operations)

Placement Tracking (Applied → Interview → Selected → Rejected)

Analytics Dashboard with charts and statistics

Role-based access (Admin, Student, HR)

How to Run
.Clone the repository

Navigate to backend → run:

bash
mvn spring-boot:run
Navigate to frontend → run:

bash
npm install
npm start
Open http://localhost:3000 for the frontend (React)
Backend runs at http://localhost:8080/api/...
